CEREBRO

O Cerebro é um projeto com o propósito de disseminar o conhecimento.

Tic Tac

Uma CLI Tool para gerenciar tarefas com base no intervalo determinado pelo usuário. O Tic tac é um método similar ao Pomodoro, com a diferença de que os minutos podem ser definidos de acordo com a necessidade da tarefa em questão.

Just - Task Runner

Um simples task runner construído com base em NodeJS e que utiliza módulos para compilar e minificar arquivos CSS, além de fazer o lint, minificar e concatenar scripts. Esta ferramenta também possui um recurso para observar a mudança dos arquivos no ambiente de desenvolvimento e suporta a sincronização entre dispositivos móveis.

Skeleton

Um projeto que fornece uma estrutura inicial para novos projetos web utilizando recursos e tecnologias atuais.

Guardian

Programa em Shell para realizar backups de segurança utilizando o poder do Rsync (somente para usuários *nix).
